Detailed Description

This is the main file for the flex sesnor portion of the RobotArm project. The program includes a task for printing to the serial monitor for trouble shooting. It also includes the arduino setup function that initializes the comunication ports and gets the tasks running. After set up, the the program also includes a loop.
